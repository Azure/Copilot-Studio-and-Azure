# Screenshots

Drop screenshots referenced from the troubleshooting pages here.

**Naming convention:** `<page-prefix>-<area>-<short-description>.png`

Examples:
- `00-toolbox-test-pane-overview.png`
- `01-basic-b3-trigger-phrases.png`
- `02-intermediate-i4-flow-run-history.png`
- `03-advanced-a3-search-explorer.png`

Keep images:
- **PNG** preferred (JPG is fine for photos).
- **Width ~1400-1800px** for readability without bloating the repo.
- **Annotated** (red boxes / arrows) when calling out a specific control.
- **Free of personal / tenant-identifying data** — blur emails, tenant ids, and secrets.
